## 개발자에게 유용한 단축키 모음 VSCode 편

### 코드 작성

코드 에디터 상에서 빠르게 작업하기 위한 단축키입니다.

- **현재 줄 복사해서 위/아래로 붙여넣기**
    - `Shift` + `Alt` + `↑` / `↓` (Mac: `Shift` + `Option` + `↑` / `↓`)

- **현재 줄 위/아래로 이동시키기**
    - `Alt` + `↑` / `↓` (Mac: `Option` + `↑` / `↓`)

- **한 줄 통째로 삭제**
    - `Ctrl` + `Shift` + `K` (Mac: `Cmd` + `Shift` + `K`)

- **같은 단어 다중 선택**
    - `Ctrl` + `D` (Mac: `Cmd` + `D`)
    - 커서를 단어에 두고 누를 때마다 다음 동일한 단어가 추가로 선택됩니다.

- **주석 처리 / 해제**
    - `Ctrl` + `/` (Mac: `Cmd` + `/`)
    - 블록 지정하고 누르면 통째로 주석 처리됩니다.

- **줄 맨 앞/뒤로 이동**
    - `Home` / `End` (Mac: `Cmd` + `←` / `→`)

- **파일 맨 위/아래로 이동**
    - `Ctrl` + `Home` / `End` (Mac: `Cmd` + `↑` / `↓`)

- **한 페이지 위/아래로 이동**
    - `Page Up` / `Page Down` (Mac: `Fn` + `↑` / `↓`)

- **커서 없이 화면만 위/아래 스크롤**
    - `Ctrl` + `↑` / `↓` (Mac: `Ctrl` + `↑` / `↓`)

- **단어 단위로 커서 이동**
    - `Ctrl` + `←` / `→` (Mac: `Option` + `←` / `→`)

<br>

### 파일 & 화면 이동

- **빠른 파일 검색 및 열기**
    - `Ctrl` + `P` (Mac: `Cmd` + `P`)

- **전체 파일에서 텍스트 검색**
    - `Ctrl` + `Shift` + `F` (Mac: `Cmd` + `Shift` + `F`)

- **에디터 창 반으로 쪼개기 (분할)**
    - `Ctrl` + `\` (Mac: `Cmd` + `\`)

- **사이드바 열고 닫기**
    - `Ctrl` + `B` (Mac: `Cmd` + `B`)

<br>

### 터미널 및 기타 기능 관리

- **내장 터미널 열기 / 닫기**
    - `Ctrl` + `~`

- **현재 열려있는 탭 닫기**
    - `Ctrl` + `W` (Mac: `Cmd` + `W`)

- **실수로 닫은 탭 다시 살리기**
    - `Ctrl` + `Shift` + `T` (Mac: `Cmd` + `Shift` + `T`)
